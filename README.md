Тестовое задание: Список из 1 000 000 элементов

Полноценное fullstack-приложение с двумя окнами для выбора и сортировки элементов через Drag & Drop.

---

Технологии:

    Бэкенд: Express.js (Node.js)
    Фронтенд: React + Vite
    UI: Простой CSS (без фреймворков)
    DnD: `@dnd-kit`
    HTTP-клиент: `axios`

---

Функционал

Левое окно:
    Отображение элементов (кроме выбранных)
    Фильтрация по ID
    Инфинити-скролл (подгрузка по 20)
    Добавление новых ID вручную

Правое окно:
    Отображение выбранных элементов
    Фильтрация по ID
    Сортировка через Drag & Drop
    Инфинити-скролл (подгрузка по 20)
    Сохранение выбора и порядка между обновлениями

Общее:
    Хранение состояния в памяти (без БД)
    Данные общие для всех пользователей
    Дедупликация и батчинг:
        Добавление новых элементов — раз в 10 сек
        Обновление выбора и сортировки — раз в 1 сек

---

Запуск проекта

1. Клонирование (если нужно)
    ```bash
    git clone <cifrovie-resheniya>
    cd cifrovie-resheniya


2. Установка зависимостей

    Установка в backend
        cd backend
        bun install

    Установка в frontend
        cd ../frontend
        bun install

3. Запуск серверов

    Откройте два терминала.

    Терминал 1: запуск бэкенда

        bash
        cd cifrovie-resheniya/backend
        bun run dev
        Сервер запустится на http://localhost:5000

    Терминал 2: запуск фронтенда

        bash
        cd cifrovie-resheniya/frontend
        bun run dev
        Интерфейс будет доступен на http://localhost:5173


                      !!Примечания!!  
    Новые ID добавляются в очередь и попадают в общий массив через 10 секунд
    Изменения выбора и сортировки отправляются на сервер с задержкой 1 секунда
    Все данные хранятся в памяти — при перезапуске сервера состояние сбросится
    Дубли при добавлении игнорируются (дедупликация через Set)
